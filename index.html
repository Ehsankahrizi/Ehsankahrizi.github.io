Assessing the Potential of Integrating Hydrological Modelling Outputs with In-situ Sensor Datasets to Improve Anomaly Detection of Streamflow Data

Utah State University, USA (2024)

This project used the ATS hydrological model to simulate streamflow in the Franklin sub-watershed and compared the results with in-situ sensor data from the Logan River Observatory. The findings suggest that modeling outputs can help distinguish between real hydrologic events and sensor anomalies, improving data quality for streamflow forecasting and environmental decision-making.
